# shiro

#### 项目介绍
此项目主要是为了回顾shiro框架
#### 软件架构
软件架构说明
就是普通的spring +shiro 

#### 安装教程

1. 将代码clone到本地
2. 运行mvn -Dmaven.test.skip=true package
3. 将.war后缀的文件copy到你的Servlet容器中，启动容器即可访问

#### 使用说明

1. 项目可以直接通过tomcat 的maven插件运行
2. 可根据情况修改spring的配置文件
3. 可将项目作为模块
